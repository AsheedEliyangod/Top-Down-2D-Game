# Top Down 2D Game

## Overview

Top Down 2D Game is an action game developed in Unity 6 where the player explores rooms, defeats enemies, and completes the level to achieve victory.

## Features

* Top-down player movement
* Camera follow system using Cinemachine
* Enemy AI patrol and chase system
* Enemy vision detection
* Projectile shooting system
* Health system
* Victory screen
* Object pooling
* Wall and obstacle collision

## Technologies Used

* Unity 6 (6000.3.9f1)
* C#
* Cinemachine
* Tilemap System

## Controls

* W A S D : Move
* Left Mouse Button : Shoot

## Project Structure

* Player Controller
* Enemy AI
* Projectile System
* Health System
* UI System
* Camera System
