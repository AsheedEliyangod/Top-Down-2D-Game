# Top Down 2D Game

## Project Overview

Top Down 2D Game is a Unity-based action game where the player navigates through a dungeon-like environment, shoots enemies, avoids enemy attacks, and survives until all enemies are defeated.

This project was developed using Unity 6 and C#.

---

## Features

- Player movement (Up, Down, Left, Right)
- Projectile shooting system
- Enemy AI with patrol and chase behaviour
- Enemy vision system
- Obstacle detection
- Enemy attack system
- Health system
- Game Over panel
- Main Menu
- Multiple enemies and patrol points

## How to Run

1. Clone the repository

git clone https://github.com/AsheedEliyangod/Top-Down-2D-Game.git

2. Open the project in Unity 6

3. Open MainScene

4. Press Play

---

## Controls

| Action | Key |
|--------|-----|
| Move Up | W |
| Move Down | S |
| Move Left | A |
| Move Right | D |
| Shoot | Left Mouse Button |
